🪔 Diwali Sales Analysis

📌 Project Overview

This project analyzes Diwali sales data using Python to identify customer purchasing behavior and business insights. The analysis helps understand which customer segments contribute most to sales and how businesses can improve customer experience and revenue.

🎯 Objectives

- Perform data cleaning and preprocessing
- Analyze customer demographics
- Identify top-performing products and categories
- Discover purchasing patterns during Diwali
- Generate actionable business insights

🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

📊 Key Analysis

- Gender-wise purchasing behavior
- Age group analysis
- State-wise sales performance
- Marital status analysis
- Occupation-based spending trends
- Product category performance

📈 Key Insights

- Identified the most profitable customer segments.
- Determined top-selling product categories.
- Analyzed regional sales distribution.
- Found patterns that can help improve marketing strategies.

📂 Files

- "Diwali_Sales_Analysis.ipynb" – Complete analysis notebook
- "Diwali Sales Data.csv" – Dataset used for analysis
- "README.md" – Project documentation

🚀 How to Run

1. Clone this repository.
2. Install required libraries:
   pip install pandas numpy matplotlib seaborn
3. Open Jupyter Notebook.
4. Run "Diwali_Sales_Analysis.ipynb".

📌 Conclusion

The project provides valuable insights into customer behavior during the Diwali season and can help businesses make data-driven decisions to improve sales and customer satisfaction.
